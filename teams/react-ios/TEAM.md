This team delivers the **Gratifire** client: **React** for product and game UI logic, and **iOS/Xcode** for packaging, signing, and store-ready builds. Implementation work targets **[github.com/slbailey/gratifire](https://github.com/slbailey/gratifire)**; this **gratifire-game-studios** repo defines agents and skills only. **Lead programmer** coordinates assignments; **react-specialist** owns React architecture; **ios-build-engineer** runs macOS-only tooling on the **remote Mac mini** via **SSH**; **ui-programmer** and **gameplay-programmer** implement screens and mechanics; **engine-programmer** (platform/runtime) covers Metro, native bridges, and runtime integration.
