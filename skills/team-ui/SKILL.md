Orchestrate **UI implementation** for Gratifire: **ux-designer** and **art-director** define flows and visuals; **ui-programmer** implements screens in **React**; **react-specialist** guards architecture and performance; **ios-build-engineer** handles native presentation edge cases. Use for menu systems, HUD, onboarding, settings, and store-safe layout (safe area, Dynamic Type where applicable).

**Remote Mac mini**: UI that depends on native behavior is validated with builds from the mini via **SSH** when developers are off-macOS.
